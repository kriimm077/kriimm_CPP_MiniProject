## 📌 Overview

A simple hand cricket game built in C++ where the user plays against the computer using random number logic.

---

## ⚙️ Run

```bash id="runx1"
g++ hand_cricket.cpp -o game
./game
```

---

## 🎮 Gameplay

* Choose batting or bowling
* Enter numbers (1–6)
* Same number = OUT
* Score and chase target
